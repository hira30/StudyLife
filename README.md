## StudyLifeの概要
自分の学習内容を記録して、可視化することで勉強の習慣化をサポートします。
ご使用の際はテストユーザーをログイン画面に表示しているので、そちらをお使いください。

## 主な機能
- 認証機能（ユーザー登録、ログイン、パスワード変更）
- お問合せフォーム
- 学習記録のCRUD
- 学習記録のグラフ表示

## 使用技術
- Python3.8.5
- Django2.2.5
- AWS EC2(Amazon Linux2)
  - Apache
  - MySQL
